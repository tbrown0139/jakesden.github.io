Introduction to xf86vidmodeproto

Package Information

  • Download (FTP): ftp://ftp.x.org/pub/individual/proto/xf86vidmodeproto-2.3.1.tar.bz2

  • Download MD5 sum: e793ecefeaecfeabd1aed6a01095174e

Installation of xf86vidmodeproto

Install xf86vidmodeproto by running the following commands: 

packagedir=xf86vidmodeproto-2.3.1
  ./configure $XORG_CONFIG

Now as the root user: 

 make install
