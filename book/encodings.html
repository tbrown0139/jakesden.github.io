encodings

Introduction to encodings

Package Information

  • Download (HTTP): http://ftp.x.org/pub/individual/font/encodings-1.0.4.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/font/encodings-1.0.4.tar.bz2

  • Download MD5 sum: 0f2d6546d514c5cc4ecf78a60657a5c1

Installation of encodings

Install encodings by running the following commands: 

packagedir=encodings-1.0.4
    ./configure $XORG_CONFIG
    make

Now as the root user: 

 make install
