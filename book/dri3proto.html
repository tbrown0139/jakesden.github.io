dri3proto

Introduction to dri3proto

Package Information

  • Download (HTTP): http://ftp.x.org/pub/individual/proto/dri3proto-1.0.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/proto/dri3proto-1.0.tar.bz2

  • Download MD5 sum: a3d2cbe60a9ca1bf3aea6c93c817fee3

Installation of dri3proto

Install dri3proto by running the following commands: 

packagedir=dri3proto-1.0
  ./configure $XORG_CONFIG

Now as the root user: 

 make install
