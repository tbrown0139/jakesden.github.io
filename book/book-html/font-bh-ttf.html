Introduction to font-bh-ttf

Package Information

  • Download (FTP): ftp://ftp.x.org/pub/individual/font/font-bh-ttf-1.0.3.tar.bz2

  • Download MD5 sum: e8ca58ea0d3726b94fe9f2c17344be60

Installation of font-bh-ttf

Install font-bh-ttf by running the following commands: 

packagedir=font-bh-ttf-1.0.3
    ./configure $XORG_CONFIG
    make

Now as the root user: 

 make install
