Installation of libXcomposite

Install libXcomposite by running the following commands: 

packagedir=libXcomposite-0.4.4
  case $packagedir in
    libXfont2-[0-9]* )
      ./configure $XORG_CONFIG --disable-devel-docs
    ;;
    libXt-[0-9]* )
      ./configure $XORG_CONFIG \
                  --with-appdefaultdir=/etc/X11/app-defaults
    ;;
    * )
      ./configure $XORG_CONFIG
    ;;
  esac
  make
  #make check 2>&1 | tee ../$packagedir-make_check.log

Now as the root user: 

 make install
