font-util

Introduction to font-util

Package Information

  • Download (HTTP): http://ftp.x.org/pub/individual/font/font-util-1.3.1.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/font/font-util-1.3.1.tar.bz2

  • Download MD5 sum: 23756dab809f9ec5011bb27fb2c3c7d6

Installation of font-util

Install font-util by running the following commands: 

packagedir=font-util-1.3.1
    ./configure $XORG_CONFIG
    make

Now as the root user: 

 make install
