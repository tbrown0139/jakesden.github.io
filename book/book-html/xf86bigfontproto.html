Introduction to xf86bigfontproto

Package Information

  • Download MD5 sum: 120e226ede5a4687b25dd357cc9b8efe

Installation of xf86bigfontproto

Install xf86bigfontproto by running the following commands: 

packagedir=xf86bigfontproto-1.2.0
  ./configure $XORG_CONFIG

Now as the root user: 

 make install
