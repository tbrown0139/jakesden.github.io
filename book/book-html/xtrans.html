  • Download MD5 sum: c5ba432dd1514d858053ffe9f4737dd8

Installation of xtrans

Install xtrans by running the following commands: 

packagedir=xtrans-1.3.5
  case $packagedir in
    libXfont2-[0-9]* )
      ./configure $XORG_CONFIG --disable-devel-docs
    ;;
    libXt-[0-9]* )
      ./configure $XORG_CONFIG \
                  --with-appdefaultdir=/etc/X11/app-defaults
    ;;
    * )
      ./configure $XORG_CONFIG
    ;;
  esac
  make
  #make check 2>&1 | tee ../$packagedir-make_check.log

Now as the root user: 

 make install
