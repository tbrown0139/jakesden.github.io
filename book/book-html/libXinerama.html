Introduction to libXinerama

  • Download MD5 sum: 9336dc46ae3bf5f81c247f7131461efd

Installation of libXinerama

Install libXinerama by running the following commands: 

packagedir=libXinerama-1.1.3
  case $packagedir in
    libXfont2-[0-9]* )
      ./configure $XORG_CONFIG --disable-devel-docs
    ;;
    libXt-[0-9]* )
      ./configure $XORG_CONFIG \
                  --with-appdefaultdir=/etc/X11/app-defaults
    ;;
    * )
      ./configure $XORG_CONFIG
    ;;
  esac
  make
  #make check 2>&1 | tee ../$packagedir-make_check.log

Now as the root user: 

 make install
