Introduction to renderproto

Package Information

  • Download (HTTP): http://ftp.x.org/pub/individual/proto/renderproto-0.11.1.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/proto/renderproto-0.11.1.tar.bz2

  • Download MD5 sum: a914ccc1de66ddeb4b611c6b0686e274

Installation of renderproto

Install renderproto by running the following commands: 

packagedir=renderproto-0.11.1
  ./configure $XORG_CONFIG

Now as the root user: 

 make install
